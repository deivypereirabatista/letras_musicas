# 🎶 Buscador de Letras & Estudo de APIs em Python

Este projeto é um laboratório prático para consolidar meus conhecimentos em Python, focando no consumo e, futuramente, na criação de APIs RESTful.

A aplicação atual é um buscador de letras de músicas que consome uma API pública e exibe os resultados em uma interface web interativa. O objetivo é evoluir esta base para um ecossistema backend completo.

## 🚀 Tecnologias Utilizadas

**Atuais:**
* **Python 3**
* **Streamlit** (Interface Web)
* **Requests** (Consumo de API Externa)

## ⚙️ Como Executar Localmente

 **Clone o repositório:**
 
   git clone [https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git](https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git)
  
  **Instale as dependências:**
  
    pip install streamlit requests
    
  **Rode a aplicação:**
  
    streamlit run app.py
